Zen l'initié
Auteur: Bernier Allan

Conseil d'utilisation:

Pour le bon fonctionnement de l'application, vous devez avoir un document contenant les fichier fournis au telechargement:
	Zen.jar
	Log.txt
	Save_0.txt
	Save_1.txt
	Save_2.txt
	Save_3.txt
	Save_4.txt
Vous ne devez en aucun cas modifier un de ces fichiers!


Ce projet contient deux versions possedent tout deux les même fonctionnalitées, une version graphique et une console.

Pour lancer le jeu en version graphique:
	Il suffit d'executer le fichier Zen.jar (double clique).
	Ou sur console de vous déplacer à l'endrois du fichier Zen.jar puis d'effectuer la commande : 
		"java -Xmx1024M -Xms1024M -jar Zen.jar"

Pour lancer le jeu en version graphique
	Lancer une console, déplacer vous à l'emplacement du jeu puis effectuer la commande : 
		"java -Xmx1024M -Xms1024M -jar Zen.jar TERMINAL"
